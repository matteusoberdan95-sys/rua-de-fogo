SANGUE NO ASFALTO - VERTICAL SLICE v1.1
========================================

Beat 'em up sombrio na Vila Esperanca (~10 min).

CONTROLES
---------
A/D: andar (duplo-tap = correr)
W/S: trocar lane
J: combo de ataque (4 golpes por estilo)
L: pistola | E: recarregar
Segurar Q: defender | Toque Q no !PARRY!: parry
K: esquiva | Espaco: pulo
R: reiniciar | M: menu (morte/vitoria)
F9: screenshot (esconde HUD)
F1: HUD debug | F2: controles alt. | F4: limpar save

OBJETIVO
--------
Atravesse a fase, ative o checkpoint no altar, derrote mini-chefe, Rain Boss e Alpha Boss.
Cuidado: pocas de lama (lento) e agua eletrica na tempestade (raio).

FEEDBACK
--------
Envie bugs e impressoes pelo canal definido pelo time.

REQUISITOS
----------
Windows 10/11 64-bit. .NET incluido no export Godot.

GERAR BUILD
-----------
dotnet build SangueNoAsfalto.csproj
Godot: Project > Export > Windows Desktop
Ou: scripts/build-demo.ps1
