# Plano De Lancamento

## Fase atual

**Demo publica tecnica** — validar combate, leitura visual, fluxo da vertical slice e feedback externo.

## Fase 1 - Demo Steam (proxima)

Objetivo: pagina Steam de demo/playtest sem prometer conteudo que ainda nao existe.

Entregas:

- build Windows demo `1.0.0-demo`;
- 5 a 7 screenshots;
- trailer curto interno;
- pagina com descricao, tags e requisitos minimos;
- pacote zip para testers.

## Fase 2 - Early access ou demo estendida

Objetivo: expandir conteudo antes de venda completa.

Possiveis entregas:

- segunda fase (Vila Santana);
- inimigos/bosses com ataques exclusivos;
- sprites finais substituindo placeholders;
- trilha sonora e SFX reais;
- achievements Steam.

## Fase 3 - Lancamento comercial

Objetivo: produto completo com campanha curta ou modos principais definidos.

## Preco sugerido (rascunho)

| Produto | Preco BRL | Observacao |
|---|---|---|
| Demo Steam | Gratis | Coleta feedback e wishlists |
| Early access | R$ 24,90 a R$ 34,90 | Se houver 2+ fases jogaveis |
| Versao 1.0 | R$ 39,90 a R$ 49,90 | Campanha completa inicial |

Esses valores sao placeholders. Revisar apos playtest, tempo de conteudo e comparacao com indies 2D similares.

## Metricas para decidir lancamento

- tempo medio para completar a demo;
- taxa de abandono antes do checkpoint;
- clareza dos controles sem tutorial extra;
- feedback sobre identidade visual brasileira;
- estabilidade do build Windows em maquinas diferentes.
